# .dotfiles

How it should like on your LINUX:
```
~(root)
│   .tmux.conf
│   .zshrc
│   .zsh_profile
└───.config
│   │   nvim
│   │   │   *.lua
│   │   alacritty
│   │   │   *
│   │   i3
│   │   │   config
│   │   i3status
│   │   │   config
│   │   personal
│   │   │   env
│   │   │   path
└───.local
│   │   share
│   │   │   fonts
│   │   │   │   Hack*.tff
```
